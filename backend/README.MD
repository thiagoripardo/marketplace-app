# NodeJS Web-Service

Para executar, primeiramente, instale os pacotes do package.json:

```bash
$ sudo npm install
$ node app.js
```
Após isto, crie um JSON no caminho: 
```
/config/config.json
```
Neste arquivo ponha credenciais do serviço da Amazon AWS. 
Para que o web service possa utilizar o DynamoDB.
